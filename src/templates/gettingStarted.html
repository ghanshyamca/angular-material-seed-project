Why Seed project?

Seed Project helps get started with Angular Material quickly. It has multiple examples commonly used in any typical project. You might edit, delete and integrate with services quickly.

It has everything around Angular Material setup already. Package Manager NPM, Common JS script loading and browserify is already setup. You don't need to spend time setting up this stuff.

Of course if you have different Tech Stack like Bower or requirejs it's not so useful after all.

How do I get started?

  1. Clone seed project from here
  2. Open command prompt at project location and perform npm install.
  3. Run the project on any local web server.
Note: As you make changes to the project run browserify main.js o- app.js

Tech Stack

  1. Angular Material

    Angular Material Library. It makes it easy to develop Material design app using AngularJS. It depends on AngularJS, Angular Aria and Angular Animate. Go To Repo

  2. NPM

    Node Package Manager - takes care installing the library in question and all it's dependencies.Go To Site

  3. Common JS

    Module loading within JavaScript made easy. NodeJS style of writing code. It works pretty well with NPM packages. Go To Wiki

  4. Browserify

    Unline node servers, browsers by default don't work with CommonJS. This makes it compatible. Go To Site
